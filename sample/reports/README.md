# Sample Scan Report

Generated by running a comprehensive xBOM scan against the DocQ&A sample project.

## Scan Command

```bash
xbom scan sample/ --skip sbom --enrich -v -o sample/reports
```

`--skip sbom` omits cdxgen (not needed to demonstrate AI-BOM/CBOM/API-BOM).
`--enrich` activates the Strands Agent (Layer 3) and CodeGraph (Layer 4) — requires `XBOM_ENRICHMENT_PROVIDER` + API key + FalkorDB running.

## CBOM Results

Cryptographic assets (JWT HS256, bcrypt, Fernet AES-128-CBC, SHA-256, HMAC-SHA256) are present in `app/auth.py`. Detection requires `semgrep` on PATH. Without semgrep the file will be empty.

```bash
pip install semgrep  # or: brew install semgrep
xbom scan sample/ --bom-types cbom -v
```
